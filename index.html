HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is important to use the heading tags appropriately to make the page accessible to users who use screen readers
  • It is important to include alt text after an image for users using a screen reader or if the link to the image breaks
  • The footer usually contains the author, copyright, contact, sitemap, and navigation.
  • The Header appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • The id attribute is more specific than the class attribute, and the class attribute is more specific than tags

GIT

  • We use the cd command to navigate to file directories.
  • We use the code . command to open the current directory in vs code
  • The git command must precede every Git statement.
  • The checkout command is to move the working branch to a new branch.
  • The -b flag creates a new branch
  • After the flag, we create the name of our new branch, it is wise to include a prefix, such as feature, to indicate what is being changed in this branch
  • We use the git status command to ensure that we are working in the new branch
  • To add code to the current branch, we use the command git add -A to indicate that we want to add all changes
  • Once we have staged files to be committed, we use the git commit command, followed by a commit message, to commit the changes
  • The commit message should describe what the commit contains
  • We can use the pull command to receive a branch's modifications into the local environment
  • It is important to always do a git pull to make sure that you have the latest version of the project on your local computer before you start to work.
  • We can use the push command to push the changes we have locally to our remote GitHub branch.
  • The git command cheat sheet can be found here.
